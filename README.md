# flutter_application_1
尝试用 flutter 实现一个简单的工具：将一个目录下的所有 xlsx、 csv 文件转换为 json 文件（提供给游戏策划使用），转换为 json 文件的作用是提供给开发使用。

工具之前已经[用 Python 实现过](https://github.com/suhanyujie/excel2Json-py)，这次用 flutter，算是一个练手项目。
